.. _get_implants:

装備インプラント取得関数
========================

APIキーを使用して、キャラクターが刺しているインプラントを取得する関数です。

使用方法
--------

関数名::

  getImplants(key, vCode, characterID)

引数の意味
    1. APIキー
    2. APIコード
    3. キャラクターID
    4. スキルのType ID

返り値
    文字列の配列(縦にずらっと並びます)

使用例
------
::

   =getImplants(12345678, "asdkfjafaksjhfasdj...", 09876543)

細かな仕様
----------

* インプラントを刺していない場合は空文字列が返ります。
